# Rule-Based Chatbot

This is a simple chatbot that responds to user input using predefined rules.

## Features

- Greeting and small-talk responses
- Name and help responses
- Date and time responses
- Exit command handling (`bye`, `exit`, `quit`)
- Fallback response for unknown input

## How It Works

The chatbot uses:

- `if-else` style rule checks
- Basic pattern matching with Python `re` (regular expressions)

## Run

1. Open a terminal in this folder.
2. Run:

```bash
python chatbot.py
```

3. Type messages and chat with the bot.
4. Type `bye` to end.
